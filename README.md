This repository demonstartes basic CRUD operations using Node.js and Express.js across five different databases:
1. MongoDB
2. Firebase 
3. MySQL
4. PostgreSQL
5. DynamoDB

Each database implementation is maintained in a separate branch for clarity and learning.

Branch Strategy: 
1. main: contains general instructions, documentation and shared resources.
2. mongodb: CRUD operations with MongoDB
3. firebase: CRUD operations with Firebase 
4. postgresql: CRUD operations with PostgreSQL
5. dynamodb: CRUD operations with DynamoDB

Getting Started:
1. Clone the repository.
2. Switch to the branch from the database you want to explore. 
3. Follow the setup instructions in that branch's README.
